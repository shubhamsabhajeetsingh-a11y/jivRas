# JivRas Web App — UI Kit

A high-fidelity click-through prototype of the JivRas grocery web app, covering the two major surfaces:

1. **Customer Storefront** — product browsing, cart, checkout
2. **Inventory Dashboard** — employee view with sidebar, product grid, table view

## Files

| File | Description |
|---|---|
| `index.html` | Interactive prototype entry point |
| `Header.jsx` | Storefront header with nav actions |
| `ProductCard.jsx` | Customer-facing product card component |
| `Sidebar.jsx` | Inventory dashboard sidebar |
| `CartPanel.jsx` | Cart summary side panel |

## Screens in prototype

1. **Products** — storefront with blurred bg, search, product grid
2. **Cart** — cart items + summary + checkout CTA
3. **Checkout** — delivery address form + order summary
4. **Inventory Dashboard** — employee sidebar + product grid
5. **Login** — auth form
